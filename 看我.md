# 钛矩TezuemLang
一个函数式编程语言


钛矩是一个由*纯python*写成的函数式编程语言  
它的语法有一点点类似于`lisp`只不过将圆括号换成了方括号

## 你好，世界！
```
[put "你好，世界！\n"]
```
钛矩中的`put`函数类似C语言中的`printf`  
虽然我没有做格式化输出

## 注释
```
// 这是一行注释
/. 这也是一行注释
   这还是一行注释
   它们都是注释./
```
钛矩中的注释以`//`开头；或以`/.`开头，`./`结尾 *（暂不支持嵌套）*

## 运算符
```
[+ 5 2]  // 加
[- 5 2]  // 减（如果只有一个参数，那就是取相反数）
[/ 5 2]  // 除
[* 5 2]  // 乘
[/- 5 2]  // 整除（除完后删去小数部分）
[/+ 5 2]  // 整除（除完后删去小数部分并向整数部分进一）
```
